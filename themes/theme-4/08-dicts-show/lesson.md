### Goal
Learn how dictionaries store related data with labels (keys).

### What You'll Learn
- Dictionaries use key-value pairs: `{"type": "octopus", "x": 200}`
- Access values: `creature["type"]`
- Store complex data structures

### Key Concept
Instead of separate lists, group related data:
```python
creature = {
    "type": "octopus",
    "x": 200,
    "y": 300,
    "size": 120
}
```

### The Power
One creature dictionary contains all its info in one place!
