## üìù Lists: Multiple Values in One Container

### Goal
Learn how lists store multiple values you can loop through.

### What You'll Learn
- Lists hold multiple values: `sizes = [80, 100, 120]`
- Access items by index: `sizes[0]` is 80
- Loop through lists with `enumerate()`

### Key Concept
Instead of separate variables, use one list:
```python
colors = [
    OceanPalette.CORAL,
    OceanPalette.PURPLE_CORAL,
    OceanPalette.STARFISH_ORANGE
]
```
