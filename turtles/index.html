Goal:

Draw a simple side-view car using basic shapes.

What you'll learn:

  • Creating a canvas
  • Drawing rectangles and circles
  • Using colors
  • Positioning shapes with coordinates

Steps:

  1. Create a canvas: can = Canvas(800, 600)
  2. Draw the car body (use rect)
  3. Draw two wheels (use circle)
  4. Add your favorite color!
üí° Hint: The canvas coordinates start at the top-left (0, 0). X increases to the right, Y increases downward.

Available Methods:

  • can.rect(x, y, width, height, fill=Color.RED)
  • can.circle(x, y, radius, fill=Color.BLACK)
  • can.line(x1, y1, x2, y2, stroke=Color.BLACK)

Colors:

Color.RED, Color.BLUE, Color.GREEN, Color.YELLOW, Color.BLACK, Color.GRAY
